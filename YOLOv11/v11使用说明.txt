## **全网最简单上手的一键运行 **官方YOLOv11** 项目运行流程**🚀️

### 第一步：打开控制台，命令行输入

```python
cd YOLOv11
```

### 第二步：训练代码，输入

```python
python train_yolo11.py
```

即可以开始训练了


### 第三步：推理代码，输入

```python
python predict_yolo11.py
```
